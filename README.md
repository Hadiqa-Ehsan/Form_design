
# 📝 Form Design

A clean and stylish web-based form design built with HTML and CSS. Perfect for contact forms, registration pages, or user input collection.

## ✨ Features

- **Responsive Layout** – Works seamlessly on desktop, tablet, and mobile
- **Input Validation** – Basic validation for email, phone, and required fields
- **Modern Styling** – Smooth shadows, rounded corners, and hover effects
- **Multiple Field Types** – Text, email, password, dropdown, radio buttons, checkboxes
- **Submit Button** – Stylish call-to-action button with hover effect
- **Error Messages** – Clear feedback for invalid inputs

## 🛠️ Technologies Used

- **HTML5** – Form structure and input fields
- **CSS3** – Styling, animations, and responsive design
- **JavaScript** – Client-side validation and interactivity

## 📁 Project Structure

```
Form_design/
├── Form.html      # Main form page

```

## 🔧 How to Run

1. Download or clone the project
2. Navigate to the project folder
3. Open `form.html` in your web browser

Or simply double-click the `form.html` file.

## 📖 Form Fields Included

| Field Type | Description |
|------------|-------------|
| Full Name | Text input for user's name |
| Email Address | Email input with validation |
| Phone Number | Contact number field |
| Password | Secure password input |
| Gender | Radio buttons (Male/Female/Other) |
| Country | Dropdown selection menu |
| Interests | Checkboxes for multiple selections |
| Message | Textarea for longer input |
| Submit Button | Submit form data |
